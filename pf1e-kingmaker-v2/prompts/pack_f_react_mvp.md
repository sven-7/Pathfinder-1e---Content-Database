# Pack F: React Character MVP

## Objective
Build modern character create/manage/sheet flow for Kairon slice.

## Tasks
1. Character library screen.
2. Character editor for slice fields.
3. Derived stats panel using `/api/v2/rules/derive`.
4. PWA shell basics and responsive tablet layout.
5. Build quick level-up flow with diff preview (`before -> after`).
6. Build compact "battle/cheat sheet" mode using short descriptions from ingested data.
7. Keep design language clean/minimal and close to Apple-like UI direction.

## Required Outputs
- Playwright smoke tests.
- Build instructions.
- Screenshot-ready flow.
- Usable tablet layout for character sheet + combat panel.
